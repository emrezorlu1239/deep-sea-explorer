# Deep Sea Explorer 🌊

An interactive deep-sea scrolling experience built with React + Vite.  
Scroll down to dive from the sunlit surface all the way to the Mariana Trench at 11,034 meters.

## Features
- Scroll-driven depth animation (0m → 11,034m)
- 5 ocean zones with unique creatures and bioluminescent effects
- Smooth background color interpolation
- Canvas-based parallax animations
- Real-time depth meter

## Zones
| Depth | Zone | Creature |
|-------|------|----------|
| 0m | Epipelagic (Sunlight Zone) | Sardines |
| 200m | Mesopelagic (Twilight Zone) | Jellyfish |
| 1000m | Bathypelagic (Midnight Zone) | Anglerfish |
| 4000m | Abyssopelagic (Abyssal Zone) | Giant Squid |
| 6500m | Hadal Zone (Mariana Trench) | Hadal Snailfish |

## Tech Stack
- React 18
- Vite
- GSAP
- HTML5 Canvas

## Run Locally
